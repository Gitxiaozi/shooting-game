# 太空射击游戏 - 图片使用说明

## 图片文件要求

请将所有图片文件放在游戏文件夹中（与 `index.html` 同一目录）：

### 射击者图片（玩家角色）
- `shooter1.png` 或 `shooter1.jpg`（射击者1）- **推荐尺寸：50x50 像素**
- `shooter2.png` 或 `shooter2.jpg`（射击者2）- **推荐尺寸：50x50 像素**

### 敌人图片
- `enemy.png` 或 `enemy.jpg`（敌人头像）- **推荐尺寸：40x40 像素**

## 支持的图片格式

游戏会自动尝试加载以下格式（按顺序）：
- `.png`
- `.jpg`
- `.jpeg`
- `.PNG`
- `.JPG`
- `.JPEG`

## 排查问题

如果图片没有显示，请检查：

1. **文件位置**：确保图片文件在 `cursor001/shooting-game` 文件夹中
2. **文件名称**：确保文件名完全匹配（不区分大小写）
   - 射击者：`shooter1` 和 `shooter2`
   - 敌人：`enemy`
3. **图片尺寸**：
   - 射击者：建议 50x50 像素（游戏会自动缩放）
   - 敌人：建议 40x40 像素（游戏会自动缩放）
4. **浏览器控制台**：按 F12 打开开发者工具，查看 Console 标签页中的错误信息
5. **文件格式**：确保图片文件没有损坏，可以在其他程序中正常打开

## 调试信息

游戏会在浏览器控制台中输出详细的加载信息：
- 成功加载时会显示：`shooter1 图片加载成功: shooter1.png` 或 `敌人图片加载成功: enemy.png`
- 加载失败时会显示：`shooter1.png 加载失败，尝试下一个格式` 或 `enemy.png 加载失败，尝试下一个格式`

如果所有格式都加载失败，游戏会使用默认图形：
- 射击者：绿色三角形飞船
- 敌人：红色方块带X标记
