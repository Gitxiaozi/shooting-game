# 快速开始指南 - 提取头像

## 当前状态
✅ 已安装 Python
✅ 已安装 Microsoft.VisualStudio.Services 插件

## 下一步操作

### 步骤1：安装Python库（必需）

打开**命令提示符**（CMD）或**PowerShell**，然后运行以下命令：

```bash
pip install opencv-python Pillow
```

**如果提示权限错误**，尝试：
```bash
pip install --user opencv-python Pillow
```

**如果pip命令找不到**，尝试：
```bash
python -m pip install opencv-python Pillow
```

或者：
```bash
python3 -m pip install opencv-python Pillow
```

等待安装完成（可能需要几分钟）。

### 步骤2：准备照片文件

1. 找到包含学生集体照的图片文件（.jpg 或 .png 格式）
2. 记住图片的完整路径，例如：
   - `C:\Users\Administrator\Desktop\class_photo.jpg`
   - 或者将图片复制到游戏文件夹：`C:\Users\Administrator\cursor001\shooting-game\`

### 步骤3：运行提取脚本

#### 方法A：使用批处理文件（最简单）

1. 打开文件管理器，进入游戏文件夹：
   `C:\Users\Administrator\cursor001\shooting-game\`

2. 双击 `提取头像.bat` 文件

3. 按照提示操作：
   - 可以直接将图片文件拖拽到命令窗口
   - 或者输入图片的完整路径

#### 方法B：使用命令行

1. 打开**命令提示符**（CMD）或**PowerShell**

2. 进入游戏文件夹：
   ```bash
   cd C:\Users\Administrator\cursor001\shooting-game
   ```

3. 运行脚本：
   ```bash
   python extract_faces.py "图片路径"
   ```
   
   例如：
   ```bash
   python extract_faces.py "C:\Users\Administrator\Desktop\class_photo.jpg"
   ```

   或者直接运行，然后输入路径：
   ```bash
   python extract_faces.py
   ```

### 步骤4：查看结果

脚本运行完成后：

1. 检查 `enemies` 文件夹（会自动创建）
   - 位置：`C:\Users\Administrator\cursor001\shooting-game\enemies\`
   - 文件：`enemy1.png`, `enemy2.png`, `enemy3.png` 等

2. 检查游戏文件夹根目录
   - 应该有一个 `enemy.png` 文件（通用敌人头像）

### 步骤5：测试游戏

1. 打开游戏：双击 `index.html` 文件
2. 点击"开始游戏"
3. 应该能看到使用提取的头像作为敌人

## 常见问题

### Q1: 提示"python不是内部或外部命令"
**解决方法：**
- Python可能没有添加到系统PATH
- 尝试使用完整路径，例如：`C:\Python39\python.exe extract_faces.py`
- 或者在安装Python时勾选"Add Python to PATH"选项

### Q2: 提示"找不到模块cv2"
**解决方法：**
- 确保已安装opencv-python：`pip install opencv-python`
- 如果使用虚拟环境，确保已激活

### Q3: 提示"未检测到人脸"
**解决方法：**
- 检查照片中是否有人脸
- 确保照片清晰，光线充足
- 尝试使用其他照片

### Q4: 提取的头像数量不对
**解决方法：**
- 脚本会自动检测人脸，可能某些人脸太小或角度问题无法检测
- 可以尝试调整照片或使用不同角度的照片
